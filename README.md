# 🍜 Ramen Shop App (C++)

Program sederhana berbasis C++ untuk simulasi pemesanan ramen di sebuah restoran.

## 🚀 Fitur

* Menampilkan menu ramen
* Menambah pesanan
* Menghapus pesanan
* Menampilkan daftar pesanan
* Checkout dengan pajak
* Sistem diskon (callback function)

## 🧠 Konsep yang Digunakan

* Struct
* Pointer
* Function Overloading
* Inline Function
* Default Argument
* Callback Function
* Dynamic Memory Allocation (`new` & `delete`)
